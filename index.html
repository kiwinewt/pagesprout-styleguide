Introduction to Themes

Themes are a way to customise the look and feel of your PageSprout site.

A theme does not have any template files. Instead, you should use the existing markup to style your theme, a la CSS Zen Garden.

Structure of a Theme

Themes live under public/themes/themename.

The two files you must include are: master.css and screenshot.jpg.

A typical theme will look like this:

  • public/
    • themes/
      • themename/
        • images/
        • stylesheets/
          • master.css
        • screenshot.jpg

Building Your First Theme

  1. Open the public/themes/ folder
  2. Create a new folder called mytheme
  3. Create a folder called stylesheets
  4. Create a new text file called master.css and save it in the stylesheets folder

You can add more stylesheets by adding the @import statement in your master.css file: 

@import 'general.css';
@import 'composition.css';

Warning: Altering Layout

While you can modify the layout in app/views/layouts/master.html.erb, beware that altering the HTML will likely break any other themes installed. Change at your own risk.

Structure of a Page

Pages are largely built around the proposed HTML 5 structure: 

<body>
  <div id="header">...</div>
  <ul id="nav">...</ul>
  <ul id="subnav">...</ul>
  <div class="article">
    <div class="section">
      ...
    </div>
  </div>
  <aside>...</aside>
  <div id="footer">...</div>
</body>
